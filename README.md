# 📊 Data Transformer SQL Project

A beginner-friendly SQL project developed using **MySQL** to practice data transformation, querying, and analysis. This project demonstrates core SQL concepts through practical examples using a sample database.

---

# 🎯 Project Objective

The objective of this project is to strengthen SQL skills by working with relational databases and performing various data retrieval and transformation operations.

---

# 🛠️ Technologies Used

- MySQL
- SQL
- XAMPP
- GitHub

# ✅ Project Tasks

- Retrieve customer and order details using different JOINs.
- Find customers with orders above the average order amount.
- Find employees earning above the average salary.
- Extract year and month from dates.
- Calculate the difference between dates.
- Format dates into different formats.
- Concatenate first and last names.
- Replace text values.
- Convert text into uppercase and lowercase.
- Remove unnecessary spaces from strings.
- Calculate running totals using window functions.
- Rank records using RANK().
- Categorize records using CASE statements.
